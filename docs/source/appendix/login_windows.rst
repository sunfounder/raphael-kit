PuTTY
=========================

Windows ユーザーの場合、SSH 接続用のアプリケーションを使用することができます。ここでは `PuTTY <https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html>`_ をお勧めします。

**ステップ 1**

PuTTY をダウンロードします。

**ステップ 2**

PuTTY を開き、左側のツリー状メニューから **Session** をクリックします。  
**Host Name (or IP address)** の入力欄に Raspberry Pi の IP アドレスを入力し、**Port** に **22** を入力します（デフォルトは 22）。

.. image:: img/image25.png
    :align: center

**ステップ 3**

**Open** をクリックします。初めて Raspberry Pi に IP アドレスで接続するときは、セキュリティの確認メッセージが表示されます。**Yes** をクリックします。

**ステップ 4**

PuTTY のウィンドウで「**login as:**」と表示されたら、  
ユーザー名に「**pi**」（Raspberry Pi のユーザー名）を、パスワードに「**raspberry**」（変更していなければ初期設定のもの）を入力します。

.. note::

    パスワード入力時に文字が画面に表示されないのは正常です。正しいパスワードを入力してください。  
    
    PuTTY の横に「inactive」と表示された場合、接続が切断されたことを意味しますので再接続してください。

.. image:: img/image26.png
    :align: center
