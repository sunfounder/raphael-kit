    # Definir pin GPIO para el bot√≥n del joystick (SW pin)

    # Configurar modo GPIO

        """
        Lee un valor anal√≥gico del canal especificado del MCP3008 (0‚Äì7)
        :param channel: N√∫mero de canal ADC (0‚Äì7)
