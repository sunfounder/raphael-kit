IoT
======================

この記事では、Raspberry Piを **Blynk** プラットフォームに接続して、面白いIoTプロジェクトを作成する方法をご紹介します。

Blynkは、個人のIoTプロジェクトから数百万台の商用接続製品に至るまで、任意の規模で電子デバイスをプロトタイプ、展開、リモート管理するために必要なソフトウェアスイートです。Blynkを使用すれば、誰でもクラウドにハードウェアを接続して、リアルタイムおよび履歴データを分析したり、デバイスを世界中からリモートで制御したり、重要な通知を受け取ったり、その他多くのことができます。
