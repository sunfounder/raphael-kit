Raspberry Pi のセットアップ
=====================================

画面を使用したセットアップ
---------------------------

画面を接続すると Raspberry Pi の操作が簡単になります。

**必要な部品**

* Raspberry Pi
* 電源アダプター
* Micro SD カード
* 画面用電源アダプター
* HDMI ケーブル
* 画面
* マウス
* キーボード

**手順**:

#. マウスとキーボードを Raspberry Pi に接続します。

#. HDMI ケーブルを使用して画面と Raspberry Pi の HDMI ポートを接続します。  
   画面が電源に接続され、オンになっていることを確認してください。

#. 電源アダプターで Raspberry Pi に給電します。数秒後に Raspberry Pi OS デスクトップが画面に表示されます。

    .. image:: img/bullseye_desktop.png
        :align: center

画面を使用しないセットアップ
------------------------------

モニターがない場合でも、リモートログインで操作可能です。

**必要な部品**

* Raspberry Pi
* 電源アダプター
* Micro SD カード

SSH を使用すると、Raspberry Pi の Bash シェルにアクセスできます。  
Bash は Linux の標準シェルで、さまざまな作業をコマンドラインで実行できます。

グラフィカルユーザーインターフェース (GUI) を利用したい場合は、リモートデスクトップ機能を使えば、ファイル管理や操作が便利になります。
